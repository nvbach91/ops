Online Point of Sale System
===========================

A javascript-written system for POS
-----------------------------------
### Powered by
 * jQuery
 * NodeJS
 * MongoDB

## Project Setup

**Install node packages**
```shell
$ npm install
```

**Run project**
```shell
$ npm start
```

or

```shell
$ gulp
```

then let NodeJS open the app in your default browser (Chrome)
